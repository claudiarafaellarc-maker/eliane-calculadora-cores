## 📝 Notas Técnicas

### Algoritmo de Mistura de Cores
A calculadora usa **média ponderada RGB** para simular mistura de pigmentos:

```
Resultado = (Pigmento1.RGB × Peso1 + Pigmento2.RGB × Peso2) / (Peso1 + Peso2)
```

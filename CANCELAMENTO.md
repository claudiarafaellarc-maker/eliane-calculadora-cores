## 🗑️ INSTRUÇÕES DE CANCELAMENTO COMPLETO

### Passo 1: Deletar Repositório GitHub

```bash
gh repo delete claudiarafaellarc-maker/eliane-calculadora-cores --yes
```

Ou acesse manualmente:
1. https://github.com/claudiarafaellarc-maker/eliane-calculadora-cores
2. Clique em **Settings**
3. Desça até **Danger Zone**
4. Clique em **Delete this repository**
5. Digite o nome do repositório e confirme

### Passo 2: Deletar Projeto do Cloudflare Pages

1. Acesse: https://dash.cloudflare.com/
2. Vá para **Pages** (no menu esquerdo)
3. Selecione **eliane-calculadora-cores**
4. Clique em **Settings** (ícone de engrenagem)
5. Desça até **Delete project**
6. Confirme a exclusão

**Resultado:** O site em https://eliane-calculadora-cores.pages.dev será removido

### Passo 3: Deletar Projeto Local

```bash
rm -rf ~/Downloads/eliane-calculadora-cores
```

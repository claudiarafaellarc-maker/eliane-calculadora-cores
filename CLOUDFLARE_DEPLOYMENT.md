# Instruções de Deploy - Cloudflare Pages

## ✅ Repositório GitHub

Seu projeto já está no GitHub:
**https://github.com/claudiarafaellarc-maker/eliane-calculadora-cores**

## 🚀 Deploy no Cloudflare Pages (3 passos simples)

### Passo 1: Acesse o Cloudflare Dashboard
1. Acesse https://dash.cloudflare.com/
2. Faça login com sua conta Cloudflare (ou crie uma se não tiver)

### Passo 2: Conectar o Repositório GitHub
1. No dashboard, clique em **"Pages"** no menu esquerdo
2. Clique em **"Create a project"**
3. Clique em **"Connect to Git"**
4. Selecione **GitHub** como provedor
5. Autentique com sua conta GitHub
6. Selecione o repositório **claudiarafaellarc-maker/eliane-calculadora-cores**
7. Clique em **"Begin setup"**

### Passo 3: Configurar Build
Na próxima página, você verá:
- **Project name**: `eliane-calculadora-cores` (ou seu nome preferido)
- **Production branch**: `main`
- **Build command**: deixe em branco (não precisa compilar, é um site estático)
- **Build output directory**: deixe em branco ou coloque `.` 

Clique em **"Save and Deploy"**

## ⏱️ Aguarde o Deploy
- Cloudflare fará o deploy automaticamente (leva ~1-2 minutos)
- Você receberá um link tipo: `https://eliane-calculadora-cores.pages.dev`

## 🌐 Configurar Domínio Personalizado (Opcional)

Se você tiver um domínio próprio (tipo `studio-eliane.com`):

1. No Cloudflare Pages, vá para seu projeto
2. Clique em **"Custom domains"**
3. Clique em **"Add custom domain"**
4. Digite seu domínio (ex: `calculadora.studio-eliane.com`)
5. Siga as instruções para configurar DNS no seu registrador de domínios

## 📋 Checklist Final

- ✅ Repositório no GitHub criado
- ⏳ Deploy no Cloudflare Pages (você fará)
- ⏳ Domínio personalizado (opcional, você fará)

## ❓ Dúvidas?

Se tiver problema, verifique:
1. GitHub está conectado à sua conta Cloudflare
2. Repositório é público
3. Branch principal é "main"

Qualquer dúvida, é só chamar!
